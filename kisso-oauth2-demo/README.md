#kisso-oauth2-demo

由于 apache-oltu-oauth2 很灵活 kisso 只做辅助支持，不重新造轮子。

kisso  =  cookie sso 基于 Cookie 的 SSO 中间件，它是一把快速开发 java Web 登录系统（SSO）的瑞士军刀。
